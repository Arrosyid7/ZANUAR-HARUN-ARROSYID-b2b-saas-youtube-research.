# B2B SaaS YouTube Content Strategy Research

## Project Overview
This repository contains comprehensive research data on tactics and content architecture specifically for B2B SaaS YouTube strategy.

## Why Choose These Experts?
The selected experts (such as Tyler Lessard and TK Kader) are authentic practitioners working directly in the software industry, not merely theoreticians.

## Folder Structure
- `/research/sources.md`: Comprehensive list of 10 experts and annotations.
- `/research/linkedin-posts/`: Collection of tactical insights from LinkedIn platform.
- `/research/youtube-transcripts/`: Full YouTube video transcripts extracted via API.
